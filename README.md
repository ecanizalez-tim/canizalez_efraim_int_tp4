EN

Dungeon Escape is a short escape game where the player must survive

inside a dungeon before time runs out.

Avoid enemies, manage your movement, and reach the exit.

Controls:

- WASD / Arrow Keys: Move

- E: Interact

- Esc: Pause

Made with Godot as a school project.
